Welcome to the Canary Project

The Canary Project is working to build an ambient monitoring solution for easy operational monitoring of a web application.

How It Works

The Canary Project integrates right into your web application with easy plug and play middleware that generates meaningful data to display on the dashboard.

Getting Started

Installing the Middleware

The middleware currently available integrates with Node.js Express servers. The package can easily be installed with the command: 

npm install --save canary-express

Once the npm package is installed the middleware can be hoooked into any application with the following lines: 

var canary_middleware = require('canary-express');

var canary = canary_middleware.canary({token: "EXAMPLE_TOKEN"});
app.use(canary);

Running the Server

Requires Go and Postgresql installed. Once go is installed you can run Run the following commands in a PostgreSQL shell 

CREATE USER gorm WITH PASSWoRD 'mypassword';
CREATE DATABASE canary.db;
To build the Go application run: 
go get ./...
go build
./canary-server
